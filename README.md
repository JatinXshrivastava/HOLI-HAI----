# Happy Holi! 🎨

An interactive **Festival of Colors** celebration website with colorful particles, animations, and splash effects.

## How to Use

1. **Open the project** – Open `index.html` in your web browser
2. **Interact with colors** – Click on the color palette dots to select your color
3. **Create splashes** – Click anywhere on the page to create colorful splash effects
4. **Enjoy animations** – Watch particles float across the background

## Files

- `index.html` – Main HTML structure
- `script.js` – Interactive effects and animations
- `style.css` – Styling and visual effects
